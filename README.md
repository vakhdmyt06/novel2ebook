# novel2ebook</br >
Novel(readlightnovels.net) to eBook Converter</br >

Requirements:</br >
pandoc</br >
gridsite-clients</br >
(Ubuntu) sudo apt install pandoc gridsite-clients -y</br >

Run:</br >
run 'bash novl2epub.sh', and follow the instructions
